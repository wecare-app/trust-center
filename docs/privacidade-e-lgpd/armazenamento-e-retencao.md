# Armazenamento e Retenção

## Armazenamento

Os dados são armazenados em ambientes seguros, utilizando práticas de proteção adequadas.

## Retenção

Os dados são mantidos apenas pelo período necessário para:

* cumprimento das finalidades contratuais
* obrigações legais

Após esse período, são excluídos ou anonimizados de forma segura.

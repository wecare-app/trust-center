# Registro de Operações de Tratamento (ROPA)

A WeCare mantém registros das operações de tratamento de dados pessoais.

Esses registros incluem:

* categorias de dados
* finalidades
* bases legais
* compartilhamentos

Os ROPAs estão disponíveis conforme nível de acesso e necessidade.

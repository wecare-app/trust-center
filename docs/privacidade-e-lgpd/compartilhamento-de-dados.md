# Compartilhamento de Dados

A WeCare realiza o compartilhamento de dados de forma limitada e controlada.

## Compartilhamento

O compartilhamento ocorre apenas quando necessário para:

* funcionamento da plataforma
* integrações com serviços externos
* cumprimento de obrigações legais

## Segurança

Todos os terceiros envolvidos seguem padrões de segurança e proteção de dados.

O compartilhamento é restrito ao mínimo necessário para a finalidade.

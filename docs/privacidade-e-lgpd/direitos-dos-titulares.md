# Direitos dos Titulares

A WeCare garante os direitos dos titulares de dados conforme a LGPD.

## Direitos

Os titulares podem solicitar:

* acesso aos dados
* correção de dados
* exclusão
* portabilidade
* revogação de consentimento

## Atendimento

As solicitações são tratadas por canais oficiais, com resposta dentro dos prazos legais.

# Dados Coletados e Finalidade

A WeCare coleta apenas os dados necessários para o funcionamento da plataforma.

## Dados Coletados

* nome
* email
* matrícula
* cargo
* celular (opcional)
* data de admissão
* data de nascimento
* liderança e área

## Finalidade

Os dados são utilizados para:

* gestão de colaboradores
* personalização da experiência
* comunicação e notificações
* funcionamento das funcionalidades da plataforma

Não realizamos coleta de dados sensíveis.

# Integrações

A WeCare oferece integração com sistemas corporativos por meio de APIs e serviços de comunicação.

## API de Colaboradores

A API permite:

* cadastro e atualização de colaboradores
* gestão de status (férias, desligamento)

## Autenticação

As integrações utilizam autenticação baseada em tokens (JWT).

## Notificações

A plataforma permite envio de notificações por diferentes canais, configuráveis pelo usuário.

## Segurança

As integrações seguem práticas de segurança, incluindo:

* autenticação
* controle de escopo de dados
* comunicação criptografada

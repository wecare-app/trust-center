# Autenticação e Controle de Acesso

A WeCare implementa mecanismos robustos de autenticação e controle de acesso.

## Autenticação

* login com credenciais seguras
* autenticação via tokens para APIs

## SSO

Suporte a Single Sign-On (SSO) utilizando SAML 2.0, com integração com:

* Microsoft Azure AD / Entra ID
* Google Workspace
* RedHat SSO
* KeyCloak

## Controle de Acesso

O acesso é baseado em perfis e permissões, seguindo o princípio do menor privilégio.

## MFA

Contas com privilégios elevados utilizam autenticação multifator (MFA).

# Segregação de Ambientes

A WeCare adota segregação de ambientes para garantir segurança e integridade dos dados.

## Ambientes

A plataforma possui os seguintes ambientes:

* desenvolvimento
* homologação
* produção

## Isolamento

Cada ambiente é isolado logicamente, evitando interferência entre testes e operação real.

## Controle de Acesso

O acesso aos ambientes é controlado conforme o perfil do usuário e necessidade operacional.

## Uso de Dados

Dados de produção não são utilizados em ambientes de desenvolvimento, garantindo privacidade e conformidade.

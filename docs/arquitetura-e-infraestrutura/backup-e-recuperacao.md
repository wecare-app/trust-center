# Backup e Recuperação

A WeCare mantém políticas de backup e recuperação para garantir a continuidade dos dados.

## Backups

* realizados diariamente
* modelo incremental
* armazenados em ambiente seguro

## Testes de Restauração

Testes periódicos são realizados para garantir a integridade dos backups.

## Recuperação

Procedimentos de recuperação permitem restaurar dados em caso de falhas ou incidentes.

## Retenção

Os backups são mantidos conforme políticas internas e requisitos legais.
